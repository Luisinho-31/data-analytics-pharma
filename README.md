# data-analytics-pharma


Portafolio de proyectos de análisis de datos, automatización e IA aplicada 
al sector farmacéutico y salud.

Cada proyecto resuelve un problema real de negocio que enfrentan distribuidoras, 
clínicas y empresas del sector: datos dispersos en Excel, reportes manuales, 
inventarios sin visibilidad y decisiones sin respaldo de datos.
